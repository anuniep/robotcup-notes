这份笔记记录了p今天从零开始通关 Linux 核心命令以及排查大版本的全过程

---

## 🛠️ 第一部分：Linux 基础文件与目录管理

### 1. 查看当前位置 —— `pwd`

* **指令：** `pwd` *(Print Working Directory)*
* **作用：** 打印出你当前身处系统中的哪一个具体文件夹。

### 2. 列出文件夹内子文件 —— `ls`

* **指令：** `ls` *(List)*
* **作用：** 把当前文件夹下的所有文件和子文件夹全部列出来。
* **💡 注意事项：** * 如果输出为空，直接换行，代表当前是个干净的空文件夹（没有消息就是好消息）
* 带有绿色背景的方块通常代表 Windows 系统映射过来的文件夹

### 3. 换文件夹（切换目录） —— `cd`

* **指令：** * `cd ~` ：跨回 Linux 的**私有目录**（路径会缩写显示为波浪号 `~`，完整路径是 `/home/pjoee`）。
* `cd 文件夹名` ：进入某个具体的子文件夹。
* **🔥 自动补全：** 输入文件夹前两三个字母后，按下键盘上的 **`Tab` 键**，系统会自动帮你补全剩下的名字

### 4. 创建文件夹 —— `mkdir`

* **指令：** `mkdir robocup` *(Make Directory)*
* **作用：** 在当前位置凭空创建一个叫 `robocup` 的全新文件夹。

### 5. 创造新文件 —— `touch`

* **指令：** `touch readme.txt`
* **作用：** 在当前文件夹里创建一个名为 `readme.txt` 的空白文本文档。

### 6. 文件的复制 —— `cp`

* **指令：** `cp readme.txt backup.txt` *(Copy)*
* **作用：** 把 `readme.txt` 复制一份，起名叫 `backup.txt`。

### 7. 物理删除 —— `rm`

* **指令：** `rm backup.txt` *(Remove)*
* **⚠️ 注意：** Linux 里没有回收站！用 `rm` 删除文件是**彻底从磁盘抹除**，无法右键还原，敲回车前必须保持清醒。

## 🌐 第二部分：高级系统、软件与网络管理

### 8. 借用管理员权力 —— `sudo`

* **用法：** 放在任何高级指令的最前面（例如 `sudo apt ...`）。
* **🔑 密码盲打：** 当系统要密码时，**屏幕上不会显示任何东西（连星号都没有）**。凭感觉盲打完密码，按回车即可

### 9. 软件管家联网对齐清单 —— `apt update`

* **指令：** `sudo apt update`
* **作用：** 让 Ubuntu 的官方软件管家（apt）去服务器下载最新的软件清单，安装任何新工具前都必须先跑这一步。

### 10. 安装新工具 —— `apt install`

* **指令：** `sudo apt install iputils-ping`
* **作用：** 从官方下载并自动安装名为 `iputils-ping`（网络测试工具）的软件。
* **💡 注意事项：** 中间如果停下来弹出 `[Y/n]`，代表询问是否确认占用空间，键盘敲一个小写 `y` 回车即可。

### 11. 网络测试 —— `ping`

* **指令：** `ping baidu.com`
* **作用：** 持续向百度服务器发送测试包，检查你的 Linux 系统联网是否正常、延迟有多高。
* **🔥 中止动作：** Linux 里的 ping 默认会**无休无止一直刷新下去**。看到弹出了几行数据、确认丢包率（packet loss）是 0% 后，必须立刻按下组合键 **`Ctrl + C`** 手动叫停

## 🛠️ 第三部分：版本排查

Ubuntu 26.04换成 **Ubuntu 24.04 稳定版**：

### 1. 检查 Linux 目前系统版本

* **指令：** `lsb_release -a`
* **核心观察项：** 查看 `Release:` 后面写的是不是 **24.04**。

### 2. 重装步骤（必须在 Windows CMD 中运行）

* **注意：** 不能在 Linux 内部自己删自己。必须先**点 X 关闭 Ubuntu 窗口**，按下 `Win+R` 输入 `cmd` 打开 Windows 命令提示符，然后依次输入：
  1. `wsl --unregister Ubuntu` *(注销掉错误的26.04)*
  2. `wsl --install -d Ubuntu-24.04 --web-download` *(联网精准抓取安装 24.04 镜像)*
